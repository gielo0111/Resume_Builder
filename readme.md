# Resume Builder

A streamlined repository designed to manage, build, and version-control professional resumes and cover letters using LaTeX. 

## Repository Structure

* **`latex/`**: Contains the core LaTeX source files (`.tex`) and templates for the resume.
* **`resume_coverletter/`**: Contains assets, templates, or documents specifically for generating matching cover letters.

## Prerequisites

To compile these documents locally, you need a working LaTeX distribution installed on your machine:

* **TeX Live** (Linux/Windows)
* **MacTeX** (macOS)
